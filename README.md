# Hermes Relay RF

Relay connection for devices in Russia.

## Как это работает
GitHub Dead Drop — асинхронный канал связи между Hermes и телефоном/компьютером в РФ, который не требует VPN.
Приёмник опрашивает commands.json, исполняет их и пишет результат.

Установка, настройка и использование — через GitHub RAW API. Не нужно ничего открывать на роутере.
